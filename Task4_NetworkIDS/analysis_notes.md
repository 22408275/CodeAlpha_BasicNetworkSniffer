# Network Intrusion Detection System (IDS)

Tool Used: Wireshark

## Objective
The goal of this project is to monitor network traffic and detect suspicious activity.

## Steps Performed
1. Installed and opened Wireshark.
2. Selected the Wi-Fi network interface.
3. Started capturing live network packets.
4. Generated traffic by browsing websites and running ping commands.
5. Applied filters such as ICMP, DNS, and TCP.

## Observations
The captured packets showed communication between the local device and external servers. ICMP packets appeared when using the ping command, demonstrating network activity monitoring.

## Conclusion
The IDS system successfully monitored network traffic and allowed analysis of packets. This demonstrates how intrusion detection systems help identify suspicious or abnormal network behavior.
